# 简介:

Koa是一款新的web框架，致力于成为web应用和API开发领域中的一个更小、更富有表现力、更健壮的基石。

利用async函数丢弃回调函数，并增强错误处理。Koa没有任何预制的中间件，可快速而愉快地编写服务端应用程序。
