# 70. 爬楼梯

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**注意：**给定 n 是一个正整数。

示例 1：

输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。
1.  1 阶 + 1 阶
2.  2 阶

1.  1 阶 + 1 阶 + 1 阶

解法: (dp) 采用滚动数组节省空间

/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function(n) {
  let p1 = 1,p2 = 1,p3;
  for(let i=2;i<n;i++){
    p3 = p1 + p2;
    p1 = p2;
    p2 = p3;
  }
  return p3;
}
